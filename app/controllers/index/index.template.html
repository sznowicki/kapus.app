Kapus.app
universal clipboard

tl;dr: set a room name, optionally protect with password. Paste on one device, copy on another; no matter where is it.

To log in to existing room, use same room name and pass.

Warning: if you don't set a password, or someone guess it, your data will leak.

Warning2: this website keeps your stuff in server memory, no database involved, no encryption. Data is not persistent. Will be destroyed after a minute.

I promise not to look into the memory, but again, it's not encrypted (yet). Just saying.

I keep no historical clipboards. Once you save, only latest state is kept in memory.

If you don't trust me, you can self-host it.
